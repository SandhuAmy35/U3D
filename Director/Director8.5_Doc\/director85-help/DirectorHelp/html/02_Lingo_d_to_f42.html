Lingo Dictionary > D-F > end case

 

end case

Syntax 

end case

Description

Keyword; ends a case statement.

Example

This handler uses the end case keyword to end the case statement: 

on keyDown
	case the key
		of "A": go to frame "Apple"
		of "B", "C" :
			puppetTransition 99
			go to frame "Mango"
		otherwise beep
	end case
end keyDown
