Properties of the newsprint shader

The #newsprint shader creates a dithering effect similar to a newspaper photograph. Use these properties to work with the newsprint shader:

Property

Access

Description

Default

name

Get and set

Name of shader

None

brightness

Get and set

Value controlling amount of white in shader

0.0

density

Get and set

Value controlling density of dots in newsprint image

45.0
