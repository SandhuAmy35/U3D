About groups

Both the local and public action behaviors' Parameters dialog boxes give you the option to assign the behavior to a group. Groups allow a single trigger to initiate actions across multiple sprites. To establish a group, simply choose a name for the group and assign that name in the Parameters dialog box of each behavior you attach to each sprite you want in the group.

Because the triggers are sent to the group name rather than to a specific sprite number, there are no reference changes to update when a sprite is moved from one Score channel to another.
