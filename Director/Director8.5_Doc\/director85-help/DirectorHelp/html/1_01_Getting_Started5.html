What this book covers

This book is an addition to the existing Director documentation and explains features that are new in Director 8.5. For information about Director features that existed in Director 8, see Using Director and the Lingo Dictionary.

Part I of this book introduces Director 3D. Its contents are topically divided as follows:

Director 8.5 tutorial overview provides a detailed tutorial that introduces the basic procedures for working with 3D cast members and behaviors.

3D basics overview provides an overview of the 3D features of Director 8.5.

Using 3D behaviors overview describes how to use the new 3D behaviors in Director 8.5.

Using 3D text overview describes how to create and work with 3D text.

The 3D cast member overview introduces the internal structure of 3D cast members.

Working with models and model resources overview describes in detail the relationship between models and model resources, and the use of modifiers, shaders, and textures in model construction.

Working with lights and cameras overview describes how to create and manipulate lights and cameras in 3D cast members.

Controlling the 3D world overview introduces the vector math operations that can be used for manipulating the locations and orientations of models and the commands and properties used to control sprites, cast members, and 3D rendering.

3D Lingo by feature overview contains all the new Lingo used to work with 3D cast members, organized by category. 3DLingo Dictionary presents these Lingo elements in standard dictionary format. This chapter is an addendum to the Lingo Dictionary for Director 8.

Part II describes in detail how to use the Shockwave Multiuser Server:

Using the Shockwave Multiuser Server and Xtra overview provides a detailed introduction to the Multiuser Server, a list of the new features in version 3.0, and practical applications and usage scenarios. This chapter also describes the Lingo scripting techniques used with the Multiuser Server.

Multiuser server-side scripting overview explains the benefits of server-side scripting and describes how to create server-side scripts and take advantage of multithreading.

Server application overview describes how to administer the Multiuser Server, including system requirements, installation, viewing application information, configuration, and troubleshooting.

Multiuser Lingo by feature overview introduces the multiuser scripting elements organized by category. Multiuser Lingo Dictionary overview presents these Lingo elements alphabetically, in dictionary format. This chapter is an addendum to the Lingo Dictionary for Director 8.

The Working with Flash 5 appendix summarizes the new features of Flash-related Lingo in Director 8.5.

The Director Support Center contains additional articles and How To examples about working with Director.

Using the RealMedia Xtra describes how to include RealMedia content in a Director movie.

Creating 3D Models for Use in Shockwave provides useful information about how to develop 3D models for use in Shockwave. This article also explains how to use the OBJ converter.

Using the Multiuser Server Debugging API explains how to use Multiuser Server debugging tools.

Using UDP with the Multiuser Server describes how the Multiuser Server works with UDP.
