Working with Flash 5 overview

The Flash Asset Xtra in Director 8.5 supports the player features of Flash 5, the latest version released prior to Director 8.5. While working in Director, you can import and link to all Flash 5 files, as well as files created in earlier versions of Flash.

In addition, advanced Director users can take further advantage of the Flash Asset Xtra to manipulate Flash sprites and cast members incorporated into Director movies. You can make many of these modifications by entering commands through the Lingo interface. These commands often, but not always, call ActionScript, Flash's scripting language. Director 8.5 supports several new Flash Asset Xtra Lingo commands.

This appendix lists the new Lingo commands and their functionality for users who have already worked with the Flash Asset Xtra. For more information about the Flash Asset Xtra, see Chapter 15 of Using Director Shockwave Studio for Director 8.
