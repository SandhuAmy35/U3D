Setting Script window preferences

To change the default font of text in the Script window and the color of various code elements, you use Script window preferences. Director automatically colors different types of code elements unless you turn off Auto Coloring.

To set Script window preferences:

1

Choose File > Preferences > Script.

2

To choose the default font, click the Font button and choose settings from the Font dialog box.

3

To choose the default color of text in the Script window, choose a color from the Color menu.

4

To choose the background color for the Script window, choose a color from the Background color menu.

5

To make the Script window automatically color certain code elements, select Enable for Auto Coloring. This option is on by default.

With Auto Coloring off, all text appears in the default color.

6

If Auto Coloring is on, choose colors for the following code elements from the corresponding color menus:

Keywords

Comments

Literals

Custom (terms you define in your own code)
