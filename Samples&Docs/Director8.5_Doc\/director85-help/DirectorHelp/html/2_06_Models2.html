Model resources

Model resources are 3D geometries defined in 3D modeling software or created in Director 8.5 Shockwave Studio with Lingo. A model is an object that makes use of a model resource's geometry and occupies a particular position and orientation within the 3D world. Model resources are viewable only when in use by a model. Several models may use the same model resource.
