File-defined model resource properties

Model resources defined by a W3D file imported into Director 8.5 or loaded via Lingo have a type value of #fromfile. File-defined resources are automatically assigned level of detail (LOD) modifier settings that allow models using those geometries to adjust their level of detail as needed, depending on the model's distance from the camera. See Level of detail (LOD) modifier properties for more information.
