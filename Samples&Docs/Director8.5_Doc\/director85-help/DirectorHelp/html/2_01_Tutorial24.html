Reset the camera

Once the camera starts moving around the world, it can be tricky for the user to return to the original camera position—unless you've included a way to reset the camera. Fortunately, Director 8.5 has a Reset Camera behavior, which you'll now apply.

1

In the Library palette, verify that 3D > Actions is selected. Drag the Reset Camera behavior to the Magic tricks sprite.

2

In the Parameters for Reset Camera dialog box, type Reset Camera in the "Which group does this behavior belong to?" text box. Then click OK.
