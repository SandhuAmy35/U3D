Motion commands

Use this command to work with motions:

Command

Description

map(motion, "bone name")

Maps the given motion into the current motion beginning at the named bone.

If no bone name is specified, the mapping begins at the root bone.

The map() command will replace any motion tracks mapped previously to the specified bone and all of its children. Motion mapping has no effect on motions that are already on a play list.
