Properties of the engraver shader

The #engraver shader gives the effect of an engraved metal surface. You can control the size and number of etched lines by adjusting the brightness and density properties, respectively. Use these properties to work with the engraver shader:

Property

Access

Description

Default

name

Get and set

Name of shader

None

brightness

Get and set

Value controlling amount of white in shader

0.0

density

Get and set

Value controlling number of lines used to shade an area

40.0

rotation

Get and set

Angle describing 2D rotational offset for engraving lines

0.0
