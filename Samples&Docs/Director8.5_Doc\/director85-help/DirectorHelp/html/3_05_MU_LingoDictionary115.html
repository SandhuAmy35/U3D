setBreakPoint()

Syntax 

scriptObject.setBreakPoint(#handlerName, whichLineNumberInHandler, enableTrueOrFalse)

Description

Multiuser Server server-side debugging command; enables or disables a breakpoint at the given line number in the given handler in the specified script object. Replace enableTrueOrFalse with TRUE (1) to turn on the breakpoint, or with FALSE (0) to turn it off.

In order for a breakpoint to be triggered, it must be it must be set in a handler that is running within a thread created with the new (thread) command. Breakpoints cannot be set in the server's default thread.
