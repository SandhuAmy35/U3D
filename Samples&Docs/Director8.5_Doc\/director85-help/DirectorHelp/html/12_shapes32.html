Setting PICT cast member properties

You use PICT cast member properties to change the names of PICT cast members and set their properties.

To view or change PICT cast member properties:

1

Select a PICT cast member and open the Property Inspector in Graphical view.

2

To view or edit the cast member name, use the Name field on the Member tab.

3

To specify how Director removes the cast member from memory if memory is low, choose an option from the Unload pop-up menu. See Controlling cast member unloading.
