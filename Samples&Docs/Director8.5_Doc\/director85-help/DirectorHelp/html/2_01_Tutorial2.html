What you'll learn

This tutorial introduces you to the 3D Xtra component of Director 8.5. This version of Director includes 3D behaviors in the Library palette that enable you to build a 3D movie without using the Lingo scripting language directly. (Lingo enthusiasts can refer to the 3D Lingo Dictionary for information on new Lingo syntax in Director 8.5 Shockwave Studio.)

The tutorial takes approximately one hour to complete. It covers 3D basics, including creating 3D text and using behaviors to rotate a model, change camera perspectives, and enable navigation. Additionally, the tutorial explains 3D concepts, which you can apply when creating your own 3D Director movies.
