3D text

You can easily create 3D text in Director 8.5. First you create 2D text exactly as in Director 8. Then you convert the text to 3D by choosing 3D Mode from the Display pop-up menu in the Text tab of the Property Inspector. Then use the 3D Text tab to manipulate the specific properties of the 3D text.

You can also manipulate the text cast member with Lingo or a behavior. For more information, see Director 8.5 tutorial overview and Using 3D text overview.
