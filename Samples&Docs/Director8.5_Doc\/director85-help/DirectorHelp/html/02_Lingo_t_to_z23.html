Lingo Dictionary > T-Z > timeoutPlay

 

timeoutPlay

Syntax 

the timeoutPlay

Description

System property; determines whether the timeoutLapsed property will be set to TRUE when the movie is paused with the pause command. When TRUE, timeouts will occur when the movie is paused. When FALSE, timouts will not occur when the movie is paused. The default value is FALSE.

This property can be tested and set.

Example

This statement sets timeoutPlay to TRUE, which tells Lingo to reset the timeoutLapsed property to 0 after a movie is played: 

set the timeoutPlay to TRUE

or 

the timeoutPlay = TRUE
